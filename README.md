## ✨ Why Oi Orbuz?

**Oi Orbuz** is built to help you **craft immersive digital experiences**. Instead of wrestling with complex keyframes, simply drop a production-ready component into your project and fine-tune it to match your brand's DNA.

> 💬 **Text Motion** · 🌀 **Dynamic FX** · 🧩 **Smart UI Components** · 🖼️ **Layered Backgrounds**

## 🚀 Pro Features

- **110+ Components** — Growing library of text animations, UI elements, and creative backgrounds.
- **Ultra-Performance** — Built to be lightweight, tree-shakeable, and smooth at 60fps.
- **Hyper Customizable** — Total control via props, Tailwind CSS classes, or raw source editing.
- **Full Compatibility** — 4 variants for every component: **JS-CSS, JS-TW, TS-CSS, TS-TW**.
- **Modern Workflow** — Optimized for direct copy-paste or CLI-based integration.

## 📦 Installation

Oi Orbuz integrates seamlessly with [shadcn](https://ui.shadcn.com/) and [jsrepo](https://jsrepo.dev).

```bash
# Example: Add the Antigravity component via CLI
npx shadcn@latest add @oi-orbuz/Antigravity-TS-TW
```
For manual setup, visit our [Installation Guide](https://oi-orbuz.netlify.app/get-started/installation).
